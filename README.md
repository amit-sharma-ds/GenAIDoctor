**MediFusion AI** is a voice-based AI doctor that analyzes speech and images to provide medical insights and predicts diseases using ML/DL models.

---

## What It Does
- Voice consultation using **Whisper + ElevenLabs STT/TTS**  
- Medical image analysis via **LLaMA 3 Vision**  
- Disease prediction for **Diabetes**, **Brain Tumor**, and **Heart Disease**  
- Fully interactive **Gradio UI** for real-time voice interaction  

---

## Key Features
- AI Doctor: interprets symptoms and images  
- ML/DL Disease Prediction Engine  
- Natural conversational voice output  
- Lightweight, real-time, responsive UI
